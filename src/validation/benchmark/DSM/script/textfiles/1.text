\section{Executive Summary}
This report is presented for the accreditation of ESP-r as a Dynamic Simulation Model (DSM) capable of producing Energy Performance Certificates and associated reports and documentation. The report details the results of all tests from the Enhanced Test Models to demonstrate ESP-r's modelling of the notional and reference buildings. All results from the actual buildings emulating the notional and reference buildings are within specified tolerances.

\section{Enhanced Test Models}
\subsection{Generation of notional and reference buildings}
The software automatically generates the notional, reference and typical model from the model built for simulation purposes.

\subsection{Infiltration Method}
Infiltration rates for all zones are determined using guidance given in CIBSE Guide A. The guide provides infiltration rates for different buildings based on pressure tests.
